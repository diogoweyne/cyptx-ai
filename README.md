# 🚀 Cyptx 

## 📖 Sobre o Projeto

O **Cyptx AI** é uma solução desenvolvida para o **Challenge FIAP x TOTVS**, com o objetivo de transformar transcrições brutas de reuniões comerciais em informações estruturadas e insights de negócio.

Utilizando técnicas de **NLP (Natural Language Processing)** e **TF-IDF**, o projeto analisa conversas, identifica padrões, destaca temas relevantes e auxilia na tomada de decisões orientadas por dados.

---

## 🎯 Objetivos

✅ Processar transcrições comerciais anonimizadas

✅ Identificar palavras-chave e temas relevantes

✅ Extrair insights estratégicos a partir de dados textuais

✅ Aplicar técnicas de NLP para análise de linguagem

✅ Apoiar decisões de negócio com base em dados

---

- 🐍 Python

- 🤖 Scikit-Learn

- 📝 NLP (Natural Language Processing)
- 🔍 TF-IDF

## 📂 Base de Dados

**ANON_nome_transcricao.csv**

📌 1.147 transcrições anonimizadas de reuniões comerciais realizadas em 2026.

📌 Dados utilizados para análise textual, extração de informações e geração de insights.

---

## ⚙️ Funcionalidades

- 📥 Leitura e tratamento de dados
- 🧹 Limpeza e pré-processamento textual
- 🔍 Extração de termos relevantes com TF-IDF
- 📊 Análise exploratória dos dados
- 📈 Visualização de resultados
- 💡 Geração de insights para apoio estratégico
